# Built-in Modules
<span class="weex-version">0.5</span>

## How to use

You can use a simply way like `require('@weex-module/name')` to access the apis of module. e.g.

```javascript
var dom = require('@weex-module/dom');
dom.scrollTo(this.$el('someIdForElement'), {
    offset: 0
});
```
